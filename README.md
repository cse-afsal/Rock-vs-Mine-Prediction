# Rock vs Mine Prediction

A Machine Learning project that predicts whether an object detected by sonar signals is a Rock or a Mine.

## 📌 Project Overview

This project uses Machine Learning algorithms to classify sonar signals into:
- Rock
- Mine

The model is trained using the Sonar dataset.

## 🚀 Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Google Colab

## 📂 Dataset

The dataset contains sonar signal readings bounced off metal cylinders (mines) and rocks.

## ⚙️ Workflow

1. Import Dataset
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction System
6. Accuracy Evaluation

## 🧠 Machine Learning Model

- Logistic Regression
